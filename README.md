# 隐私政策弹窗

鸿蒙(HarmonyOS)实现隐私政策弹窗，点击用户协议，不会被弹窗遮住，如果用系统自定义弹窗，因为弹窗的显示层级一直在最上层，点击用户协议后，router跳转到Web页面，弹窗会覆盖Web页面
